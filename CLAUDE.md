# Daily Report · 日报

## Design Context

- **Register:** brand — 信息展示页，设计即产品
- **North Star:** "The Morning Briefing" — 温暖的晨间简报，金融信息与人文温度的交汇
- **Color Strategy:** Full palette — primary (editorial ochre, hue ≈ 39°) / accent (deep data blue) / rise (warm red) / fall (calm teal)
- **Typography:** Serif display + sans body — 衬线标题传递编辑权威，无衬线正文保证数据可读
- **Motion:** Responsive — 平滑过渡 + 滚动反馈，像翻页
- **Anti-reference:** 冷淡数据墙、SaaS模板、奶油色背景
- **Full spec:** See [PRODUCT.md](./PRODUCT.md) and [DESIGN.md](./DESIGN.md)
